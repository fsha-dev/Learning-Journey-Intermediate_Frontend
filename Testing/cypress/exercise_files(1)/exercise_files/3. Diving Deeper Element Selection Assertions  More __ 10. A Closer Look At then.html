In the previous lecture, you learned about then().

The then() method can be chained after other Cypress queries to get direct access to the "subjects" of those queries.

For example, the subject of get('form') would be the selected form element(s).

Therefore, the concrete value received by the function passed to then() depends on the query before then(). Later in the course, you'll also see examples where then() does not receive an element but other kinds of objects and values.

When receiving an element (e.g., as it's the case for get('form').then(el => ...)), you can inspect that element inside of the function passed to then().

As mentioned in the previous lecture, you actually get a "wrapper object" around the DOM element(s) that was / were selected.

However, you can also get (even more) direct access to the actual DOM element(s) via el[0] (for the first selected element), el[1] (for the second selected element, if available) etc.
