# JavaScript Practice Projects

## Overview
This repository contains multiple **pure JavaScript projects** created for practicing frontend development concepts such as:

- DOM manipulation
- Event handling
- Timers
- UI state management

Each project is independent and has its own documentation.

## Projects Included
1. Typing Speed Test – Version 1
2. Typing Accuracy Test – Version 2
3. Data Table (Work in Progress)

## Tech Stack
- JavaScript (Vanilla)
- HTML
- CSS

## Purpose
These projects were built to:
- Improve JavaScript fundamentals
- Practice UI logic without frameworks
- Experiment with real-time calculations (WPM, accuracy)

## Notes
Some features are unfinished due to limited internet access for external APIs.
