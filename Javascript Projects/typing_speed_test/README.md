# Typing Speed Test – Version 1

## Description

This project is a simple English typing speed test application.
It measures the user's typing speed in **Words Per Minute (WPM)** while the user types a scrolling line of words.

The timer starts automatically when the user begins typing.

## Features

- Automatic timer start on first keystroke
- Real-time WPM calculation
- Scrolling word line while typing
- Try Again button to restart the test
- Typing input and displayed text are synchronized

## Tech Stack

- JavaScript (Vanilla / Pure JS)
- CSS (for styling)
- HTML

## How It Works

- The user starts typing
- Timer starts on the first key press
- Words scroll vertically as the user types
- WPM is calculated based on typed words and elapsed time
- User can restart the test using the "Try Again" button

## How to Run

1. Open `index.html` in a browser
2. Start typing to begin the test

## Limitations

- English language only
- No accuracy calculation
- No user history or results saving
