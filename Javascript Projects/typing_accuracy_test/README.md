# Typing Accuracy Test – Version 2

## Description

This is an improved version of the typing test application with a focus on **accuracy**.
The typing area is separated from the displayed sentence, and a pointer indicates the current character being typed.

Each test runs for exactly **1 minute**, during which one sentence is shown.

## Features

- Separate typing input and sentence display
- Character pointer to track typing position
- One sentence per 1-minute session
- Words Per Minute (WPM) calculation
- Typing accuracy percentage
- Total mistake counter
- Sentence randomization

## Tech Stack

- JavaScript (Vanilla / Pure JS)
- CSS
- HTML

## How It Works

- A sentence is displayed to the user
- The user types in a separate input area
- A pointer highlights the current character
- The test runs for 1 minute
- At the end, WPM, accuracy, and total mistakes are shown
- Use Try Again button for another typing test

## Variables Tracked

- WPM
- Accuracy (%)
- Total Mistakes

## How to Run

1. Open `index.html` in a browser
2. Start typing to begin the test

## Limitations

- Fixed time duration (1 minute)
- Single sentence per test

## Future Improvements

- Multiple difficulty levels
- User result history
- Show error words below typing words after 1 minute
