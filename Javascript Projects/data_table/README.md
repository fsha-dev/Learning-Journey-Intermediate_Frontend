# Data Table Project

## Description

This project is a JavaScript-based data table with theme toggling functionality.
Further development is currently paused due to internet restrictions affecting API access.

## Current Features

- Table layout
- Light/Dark mode toggle

## Planned Features

- Fetching fake user data from an API
- Pagination
- Sorting
- Search/filter functionality

## Tech Stack

- JavaScript (Vanilla / Pure JS)
- CSS
- HTML

## Development Status

⚠️ Work in progress

Development is temporarily halted due to internet blackouts in Iran, which prevents fetching data from external APIs.

## How to Run

1. Open `index.html` in a browser
2. Toggle between light and dark modes

## Notes

Once API access is available, data fetching and table features will be implemented.
